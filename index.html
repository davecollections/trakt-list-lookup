    Nuvio import

    Import Nuvio JSON

    Nuvio import flows may change while direct Trakt source support is being validated.

    Web

    1. Log in to Nuvio and select the right profile.
    2. Open Collections, then Import.
    3. Select the downloaded JSON file.
    4. Choose Add as new, Merge, or Overwrite.
    5. Confirm Add collections.

    TV app

    1. Open Nuvio and choose a profile.
    2. Open Addons, then Collections.
    3. Select Import.
    4. Choose From File or From URL.
    5. Load the JSON and confirm import.
